# Commun Axiom <img src="CommunAxiom.png" style="height: 1em" /> - Commons Ecosystem <img src="Commons.png" style="height: 1em" /> 

### The data collaboration software ecosystem at the center of the Commun Axiom software suite.

<br/>

## In this Repo

This repository contains all the logic required to support the two level network known as the Comax - Commons Network. Its purpose is to facilitate data collaborations for small compagnies and non-profits by simplifying and streamlining processes around BI, data governance, automation, api integration, microservices implementation etc. 

The Commun Axiom suite also aims at proving that decentralization of complexe platforms for voluntary hosting is possible. 

<br/>

## Technology stack

- Aspnet Blazor
- Aspnet Web Api
- Orleans
- Kubernetes / helm
- Mariadb (Litedb for development)
